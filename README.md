# OB UI Test Project

## Installation

App requires [Node.js](https://nodejs.org/) v10+ to run.

Run app:

> DEV environment

```sh
cd ob-ui-test
yarn
yarn dev

```

> PROP environment

```sh
cd ob-ui-test
yarn stage

```

> E2E Test

```sh
cd ob-ui-test
yarn dev
yarn cy:e2e

```
